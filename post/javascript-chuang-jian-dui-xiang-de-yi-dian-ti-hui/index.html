JavaScript创建对象的一点体会

2021-01-12 1 min read

JS创建对象的形式有很多，在一般的情况下，你可以通过var 对象={属性名:属性值,属性名:属性值};来创建一个对象。
也也可以通过var obj=new Object();obj.name=xiaolv;创建对象。
另外，你甚至可以使用工厂方法，就比如：
var obj=myobj();
function myobj(name,age,gender)={
var obj=new Object();
obj.name=name;
obj.age=age;
obj.gender=gender;
return obj;
}

也可自行创建构造函数，使用Function+函数名即可创建。
